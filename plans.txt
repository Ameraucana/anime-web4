	1. if !Object.keys(fileObj).includes(anilistName), add anilistName 		   to fileObj with no value

	1. if !apiData.includes(name), remove name from fileObj
5. stringify fileObj and write it to names

3. for name of Object.keys(fileObj)
	1. if aniName === name, fileObj[aniName] = userSpec
	2. break
